# 头文件命名约定

- C++旧式风格，以`.h`结尾，C++程序可以使用，比如：`iostream.h`
- C旧式风格，以`.h`结尾，C和C++都可以使用，比如：`math.h`
- C++新式风格，没有扩展名，C++程序可以使用，使用`namespace std`，比如：`iostream`
- 转换后的C，加上前缀c，没有扩展名，C++程序可以使用，可以使用不是C的特性：`namespace std`，比如：`cmath`
